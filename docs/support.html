Frequently Asked Questions

Is my data sent to MOM Shield?

No. In the current version, MOM Shield processes content locally in your browser. There is no provider-operated masking server.

Why was a PDF blocked?

PDF files are converted to text and then masked. If a PDF has no usable text layer, is encrypted, or cannot be safely extracted, MOM Shield blocks it instead of passing through the original file unchanged.

Does MOM Shield support OCR for scanned PDFs?

No. Scanned or image-only PDFs without a usable text layer are not OCR-processed in the current version.

Why was a value not masked?

MOM Shield uses local detection rules and context validation. Depending on wording and formatting, false negatives or false positives are possible.

Why can I not unmask a response?

Unmasking requires active local history. If History is off or strict privacy mode is enabled, automatic response unmasking is disabled.
