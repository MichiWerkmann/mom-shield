2. What MOM Shield Does

MOM Shield - Privacy Masking for AI is a browser extension that masks sensitive data in chat inputs and supported upload files locally in the browser before the content is passed to a supported AI chat website.

Supported target sites currently include ChatGPT, Claude/Anthropic, Gemini, and Grok Web.

3. Data Processed by MOM Shield

MOM Shield may process the following data locally when users enter it into a supported page or select it for upload:

  • text entered into chat input fields
  • contents of supported upload files, such as txt, md, csv, json, xml, html, log, docx, xlsx, and text-based pdf files
  • local extension settings, such as enabled status, upload masking setting, strict privacy mode, and history mode
  • locally generated masking placeholders and mappings if unmask history is enabled

This content may contain personal or sensitive information, such as names, email addresses, phone numbers, financial data, identifiers, credentials, or document contents.

4. Local Processing

MOM Shield processes content locally in the user's browser.

  • masking is performed on the user's device
  • content selected for masking is not sent to a server operated by the extension provider
  • the current version does not use a provider-operated cloud service for content processing

5. Purpose of Processing

  • detecting sensitive data in text inputs and files
  • replacing detected values with masking placeholders
  • optionally restoring masked values locally if unmask history is enabled
  • storing local settings so the extension works as configured by the user

6. Legal Basis

Where GDPR applies, processing is generally based on Article 6(1)(b) GDPR where processing is required to provide the functionality requested by the user, and Article 6(1)(f) GDPR where there is a legitimate interest in data security, confidentiality, and safer use of AI services.

7. Storage and Retention

MOM Shield stores data only in the user's local browser environment. This may include settings in chrome.storage.local and optional local unmask history if enabled by the user.

Retention depends on the user's configuration, browser environment, and extension usage. Data may be removed when the extension is uninstalled or local browser data is cleared.

8. Sharing With Third Parties

MOM Shield does not send processed content to the extension provider.

Content may be passed to the AI chat website chosen by the user after local processing. Further processing by the selected AI chat website is governed by that provider's own privacy policy.

9. No Sale, Advertising, or Tracking Use

  • MOM Shield does not sell user data
  • MOM Shield does not use processed content for advertising
  • MOM Shield does not use processed content for creditworthiness, lending, or scoring purposes
  • MOM Shield does not operate tracking for masked content

11. Security

MOM Shield is designed to process data locally and sparingly. Users should note that overall security also depends on their browser environment, target website, device security, and general browsing behavior.
